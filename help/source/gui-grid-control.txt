= The grid control =
The grid control enables and disables the snappable \ref(feat-grid.txt,Grid) and adjusts the grid-spacing.

||\image(gui-grid-control-disabled.png)||\image(gui-grid-control-enabled.png)||
|| When disabled, the control is collapsed. || When enabled, the spacing can be adjusted ||

To enable the grid for an image, click the grid button. This reveals
the grid spacing controls.

||\image(icon-info.png)|| The spacing digit can be dragged for quick-adjustment: ||
\image(gui-grid-control-drag.png)
