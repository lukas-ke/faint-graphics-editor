= Constrained drawing =
Many tools can be "constrained". For example, line drawing can be
constrained to certain angles, and rectangle drawing can be
constrained to squares. Selection movement can be constrained along
the horizontal or vertical axes.

To constrain, hold shift while drawing or adjusting an object or selection.\br
