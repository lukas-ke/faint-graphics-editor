= \graphic(tool-tape-measure.png) Tape measure =
label:tool_tape_measure
The tape measure tool is used to read out measurements from an image.

The tool provides either pixel measures or
\ref(feat-calibration.txt,calibrated units), and provides a convenient
alternative to \ref(ref-text-expressions.txt,text-expressions) for viewing
a length measure.

*Settings*
||\image(tape-measure-drop-down.png)||Unit of measurement||
||\graphic(tape-measure-line.png)\graphic(tape-measure-triangle.png)||Show measurements for one line or all sides of a triangle||
